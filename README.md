# sales-analysis
## id:CITS4174

This project analyzes supermarket sales data using Python. It explores sales performance, customer behavior, and payment methods through data analysis and visualizations.

## Technologies Used

- Python
- Pandas
- Matplotlib
- Google Colab

## Features
- Load the supermarket sales dataset
- Explore the dataset
- Check for missing values
- Analyze total and average sales
- Create charts and graphs
- Save the analyzed dataset

## Analysis Performed

- Total Sales
- Average Sales
- Highest Sale
- Sales by Branch
- Sales by City
- Sales by Genders

## Visualizations
- Sales by Branch (Bar Chart)
- Payment Method Distribution (Pie Chart)
- Sales Distribution (Histogram)
